TEACHER NOTES
Graph Transformations and Curve Sketching

Purpose

This tool supports classroom modelling of parent graphs, transformations of functions, and curve sketching from key features.

Teaching Focus

  • Use key points and invariant points before sketching.
  • Separate horizontal changes from vertical changes.
  • State intercepts, asymptotes, turning points, domain, and range where relevant.
  • Compare the equation with the visible change in the graph.

Common Misconceptions

  • Thinking f(x - a) moves the graph left instead of right.
  • Using a vertical stretch rule for a horizontal stretch.
  • Sketching a curve without first finding intercepts or asymptotes.
  • Applying combined transformations in the wrong order.
