Teacher Instructions

Purpose

This tool supports practice with the standard normal distribution, standardising any normal random variable, and using tables or a calculator to find probabilities and inverse values.

Level Guide

  • Level 1: Standard normal probabilities, Φ(z), complements, intervals, and inverse z-values.
  • Level 2: General normal distributions, Z = (X - μ) / σ, reverse cut-offs, and absolute-value probability statements.
  • Level 3: Finding unknown cut-offs, means, standard deviations, and pairs of parameters from probability statements.

Teaching Notes

  • Students should identify the relevant tail, interval, or cut-off before using tables or technology.
  • Worked steps deliberately show the standardisation and the table lookup separately.
  • For inverse questions, encourage students to decide whether the probability is left-tail or right-tail before finding z.
  • For parameter questions, set up x = μ + zσ and solve the resulting equation or pair of equations.

Common Mistakes

  • Using σ² instead of σ when standardising.
  • Forgetting that P(Z > z) = 1 - Φ(z).
  • Using the wrong sign of z for a lower-tail probability below 0.5.
  • Treating |X - a| > b as one inequality instead of two tails.
