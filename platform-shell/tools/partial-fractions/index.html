Teacher Instructions

Topics Covered

  • Partial fractions with distinct linear factors.
  • Partial fractions with a repeated linear factor.
  • Improper rational expressions requiring division first.
  • Partial fractions with a linear factor and an irreducible quadratic factor.

Teaching Notes

  • Start by writing the correct form of the decomposition before solving for constants.
  • Substitute convenient x-values when linear factors are distinct.
  • Use equating coefficients when substitution alone is not enough.
  • For improper fractions, divide first, then decompose the remainder fraction.
