📊 Teacher Instructions

Getting Started

This tool helps students practice statistical measures of center and spread through structured progression.

Level Guide

  • Level 1 - Individual Measures: Calculate each measure separately
  • Level 2 - Combined Measures: Find mean, median, and mode together
  • Level 3 - Comparing Measures: Interpret and compare different measures
  • Level 4 - Missing Values: Find missing numbers when mean is given

Key Formulas

Mean = Sum of all values ÷ Number of values
Missing value = (Mean × Number of values) - Sum of known values
Median = Middle value when arranged in order
Mode = Most frequently occurring value
Range = Highest value - Lowest value

Teaching Strategies

  • Use the calculation boxes to show step-by-step work clearly
  • Color-coded data helps students identify which values to work with
  • Emphasize the ordering process for finding median
  • Show the relationship: Sum = Mean × Count

Common Student Mistakes

  • Forgetting to order data before finding median
  • Confusing mean and median concepts
  • Not accounting for frequency in repeated values
  • Errors in algebra when finding missing values
