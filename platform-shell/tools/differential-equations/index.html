Differential Equations

Overview

This tool generates structured practice for separable differential equations, first-order linear differential equations, second-order constant coefficient equations, and modelling problems.

Level Guide

  • Level 1: First-order separable equations, including initial conditions.
  • Level 2: First-order linear equations using integrating factors.
  • Level 3: Second-order constant coefficient equations using the auxiliary equation.
  • Level 4: Modelling with growth, decay, cooling, and mixing equations.

Teaching Focus

  • Separate variables before integrating.
  • Use an integrating factor for equations in the form y′ + P(x)y = Q(x).
  • Form the auxiliary equation for second-order linear equations.
  • Apply initial or boundary conditions only after the general solution has been found.
