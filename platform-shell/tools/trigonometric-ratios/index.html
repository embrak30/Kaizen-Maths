Trigonometric Ratios Guidance

Overview

This tool focuses on right-triangle trigonometry using sine, cosine, and tangent. Sine Rule, Cosine Rule, arcs, and sectors are separate topics.

Level Guide

  • Level 1: Find missing sides in right triangles using SOH-CAH-TOA.
  • Level 2: Mixed side problems using sin, cos, and tan with varied units.
  • Level 3: Use inverse trig to find missing angles.

Key Formulae

  • SOH: sin θ = opposite / hypotenuse.
  • CAH: cos θ = adjacent / hypotenuse.
  • TOA: tan θ = opposite / adjacent.

Common Misconceptions

  • Labelling opposite and adjacent from the wrong acute angle.
  • Using the hypotenuse as the side touching the angle rather than the side opposite the right angle.
  • Multiplying when the unknown is in the denominator.
  • Forgetting inverse trig when finding an angle.
