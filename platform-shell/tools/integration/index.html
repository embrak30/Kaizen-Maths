Teacher Instructions

Overview

This tool supports teaching integration as reverse differentiation, beginning with indefinite integrals, moving to definite integrals, and finishing with the trapezium rule.

Level Guide

  • Level 1: Indefinite integrals, including fractional and negative powers.
  • Level 2: Definite integrals using antiderivatives and bounds.
  • Level 3: Trapezium rule from functions and tables.

Common Misconceptions

  • Forgetting + C for indefinite integrals.
  • Treating 1/x like an ordinary power instead of using logarithms.
  • Forgetting to double the interior ordinates in the trapezium rule.
