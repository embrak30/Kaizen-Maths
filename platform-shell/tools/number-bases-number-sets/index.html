Number Bases & Number Sets

Overview

This tool supports number-set classification and number bases. It is useful for CSEC, GCSE, IGCSE, and early secondary number work.

Set Convention Used

  • Natural numbers, N: positive counting numbers, 1, 2, 3, ...
  • Whole numbers, W: 0, 1, 2, 3, ...
  • Integers, Z: negative and positive whole numbers, including 0.
  • Rational numbers, Q: numbers that can be written as a fraction of integers.
  • Real numbers, R: rational and irrational numbers.

Level Guide

  • Level 1: classify numbers and reason about number sets.
  • Level 2: convert between denary and other bases.
  • Level 3: add, subtract, multiply, and solve simple unknowns in bases.
  • Level 4: mixed exam-style questions.

Teaching Focus

  • For bases, expand using place value: digits multiply powers of the base.
  • For denary to another base, use repeated division and read remainders upwards.
  • For number sets, ask for the smallest set first, then list any larger sets it belongs to.
