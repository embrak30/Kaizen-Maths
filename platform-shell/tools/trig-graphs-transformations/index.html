TEACHER NOTES
Trigonometric Graphs and Transformations

Purpose

This tool supports teaching sine, cosine, and tangent graphs using key points, amplitude, period, midline, phase shift, and asymptotes.

Teaching Focus

  • Build standard sine, cosine, and tangent graphs from key points.
  • Connect coefficient changes to amplitude and period.
  • Show phase shifts using the starting point of a cycle.
  • Mark tangent asymptotes before drawing branches.

Common Misconceptions

  • Using amplitude for tangent graphs.
  • Thinking sin(x - a) shifts the graph left instead of right.
  • Using 360b instead of 360 ÷ b for sine and cosine period.
  • Drawing tangent as a continuous curve across an asymptote.
