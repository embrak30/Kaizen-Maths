Proof by Induction

Overview

This Further Maths tool gives structured practice in mathematical induction for sums, divisibility, inequalities, recurrence relations, and matrix powers.

Induction Structure

  • Base case: show the statement is true for the first value.
  • Assumption: assume the statement is true for n = k.
  • Induction step: prove the statement for n = k + 1.
  • Conclusion: state that the base case is true and that P(k) implies P(k + 1), so the statement is true for every integer in the stated range.

Teaching Focus

  • The induction step must use the assumption.
  • A proof should show the algebra, not simply say that the next case follows.
  • Students should finish with a complete conclusion.
