Teacher Instructions

Getting Started

This tool builds fluency with two-stage tree diagrams, independent events, conditional probability, and reverse conditional probability.

Level Guide

  • Level 1: Independent two-stage events, complements, exactly one success, and at least one success.
  • Level 2: Conditional probabilities, without-replacement contexts, and using complete tree diagrams.
  • Level 3: Reverse conditional probability, total probability, and interpreting tree paths.

Teaching Focus

  • Probabilities on branches leaving the same point must add to 1.
  • Multiply along a path to get an intersection probability.
  • Add separate paths when the wording says "or".
  • For reverse conditional probability, divide the desired path by the total probability of the condition.

Common Mistakes

  • Adding along branches instead of multiplying.
  • Forgetting probabilities change when drawing without replacement.
  • Missing one of the two "exactly one" paths.
  • Using P(A and B) when the question asks for P(A | B).
