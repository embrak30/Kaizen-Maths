TEACHER INSTRUCTIONS
Elementary Maths Playground

How to Use

  • Choose an activity mode, then generate a new visual task.
  • Keep the answer hidden while pupils discuss or write their responses.
  • Reveal the answer only when the class is ready to check.

Activity Modes

  • Number Bonds: Part-part-whole tasks with a missing part.
  • Place Value: Base-ten blocks for reading hundreds, tens, and ones.
  • Arrays: Multiplication arrays using colourful counters.
  • Fractions: Shaded fraction bars with the answer hidden.
  • Clock Challenge: Analogue clock reading tasks with hidden answers.
  • Money: Coin and note counting, plus make-the-amount prompts.
  • Shape Sorter: Count, name, and sort shapes by simple properties.
  • Missing Signs: Complete number sentences by choosing the operation sign.

Teacher Flow

Ask pupils what they notice, collect answers, invite explanation, then reveal. This keeps the activity playful while still supporting mathematical language.
