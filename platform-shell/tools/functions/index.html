Functions Guidance

Overview

This tool develops function notation from substitution into functions, through substituting expressions and composing functions, to finding inverse linear functions.

Level Guide

  • Level 1: Evaluate linear and quadratic functions at numerical inputs.
  • Level 2: Substitute algebraic expressions into functions and form composite functions.
  • Level 3: Find and use inverse functions for linear functions.

Key Ideas

  • Function notation means replace every x in the rule by the input.
  • For f(5x - 1), the whole expression 5x - 1 replaces x.
  • For f(g(x)), first write g(x), then substitute it into f.
  • To find an inverse, write y = f(x), swap x and y, then solve for y.

Common Misconceptions

  • Only substituting part of an expression input.
  • Forgetting brackets around inputs such as 5x - 1.
  • Expanding squared brackets incorrectly in quadratic functions.
  • Reversing operations in the wrong order when finding an inverse.
