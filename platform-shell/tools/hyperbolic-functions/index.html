Hyperbolic Functions

Overview

This Further Maths tool covers hyperbolic definitions, identities, equations, calculus, and inverse hyperbolic functions.

Level Guide

  • Level 1: Definitions, exact values, and identities.
  • Level 2: Hyperbolic equations using exponential form and inverse functions.
  • Level 3: Differentiation, integration, product rule, and second derivatives.
  • Level 4: Inverse hyperbolic functions, logarithmic forms, and derivatives.

Key Facts

  • sinh x = (ex - e-x) / 2.
  • cosh x = (ex + e-x) / 2.
  • cosh2x - sinh2x = 1.
  • tanh x = sinh x / cosh x.
