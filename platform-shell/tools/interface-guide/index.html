Teacher Guide

How to use Kaizen Maths in the classroom

Kaizen Maths is designed to feel like a virtual textbook with unlimited fresh practice. Choose a topic, select a level and question type, project a board-ready set, reveal answers or worked steps, and build worksheets from the same generators when you need printed practice.

1

Start at the dashboard

Use the dashboard links for the Tool Library, Worksheet Builder, and this guide.

2

Find a topic

Search like a textbook or browse by Algebra, Number, Geometry, Statistics, and Classroom Tools.

3

Project practice

Open Classroom View, choose a level and question type, then generate a fresh set.

4

Model or print

Use One Example for teaching, Practice Set for students, or the Worksheet Builder for handouts.

1. Dashboard

Use the dashboard as the launch point

The dashboard is the front door of the site. It gives teachers a quick route into the main workflows: building a worksheet, reading this guide, opening the full tool library, or jumping into a subject collection.

  • Create Worksheet opens the worksheet builder for printed or downloadable practice.
  • How to Use This Site opens this illustrated guide.
  • Open Tool Library shows every available topic generator.
  • Subject collections help you browse topics like a textbook chapter list.
For new users, start with the guide once, then use the Tool Library as your daily teaching route.
2. Tool Library

Search and browse topics

The Tool Library is the main index of the virtual textbook. Use the search bar when you know the exact topic, or use the collection links in the sidebar to browse by curriculum area.

  • Search for terms such as functions, circle, fractions, or trigonometry.
  • Use category filters to narrow the library.
  • Open a topic card to see a short description and teacher guidance.
The library is meant to be compact and searchable, so it can grow without becoming hard to navigate.
3. Topic Page

Read the topic overview before projecting

Each tool has a topic page before the classroom view. This page explains what the generator covers and provides teacher guidance without crowding the projected question screen.

  • Back to Library returns to the searchable index.
  • Classroom View opens the generator for projection.
  • Teacher Guidance summarises the topic coverage and standards alignment.
Use the topic page when planning. Use Classroom View when teaching.
4. Classroom View

Generate board-ready question sets

The classroom view is designed for projection. Choose a level, choose a question type, then generate questions. Most algebra tools show five questions. Diagram-heavy tools usually show four so the whole set remains visible on the board.

  • Level tabs move through the teaching sequence for the topic.
  • Question type lets you focus on a specific skill inside the level.
  • New creates another fresh set of the same type.
  • Answers toggles quick answers.
  • Steps toggles worked solutions. Steps may scroll, but the question set is designed to fit the board.

Use One Example for modelling

The display switch has two choices: Practice Set and One Example. One Example shows a single question so the teacher can model the method before students practise.

Return to Practice Set for student work

Click Practice Set to return to the normal full set. The active choice is highlighted, so the current mode is always visible.

5. Worksheet Builder

Create mixed worksheets from the generators

The Worksheet Builder uses the same topic levels and question types as the classroom tools. Teachers can combine different topics into one worksheet and generate a separate answer key.

  • Add questions from one topic or several topic areas.
  • Choose the level, question type, and number of questions for each section.
  • Use quick-add counts for common worksheet sizes.
  • Reset the worksheet selections if you want to start again.
  • Generate a student copy and a separate teacher answer key.
The worksheet builder is best for consolidation, cover work, homework, or independent practice after a modelled example.
Suggested Lesson Rhythm

A simple classroom workflow

  1. Choose the topic: Find the tool in the library or subject collection.
  2. Model one example: Use One Example mode to talk through the method.
  3. Generate a practice set: Switch to Practice Set and let students attempt the visible questions.
  4. Use the timer if helpful: Two minutes for quick fluency, five minutes for longer tasks.
  5. Reveal answers: Check responses quickly without showing the full method too early.
  6. Reveal steps: Discuss the worked solution once students have committed to an approach.
  7. Generate again: Use New for another set at the same level or change question type.
  8. Print follow-up: Use the Worksheet Builder to create independent practice from the same topics.

The key idea

Kaizen Maths is built around teacher control: one example when you are modelling, a compact set when students are practising, worked steps when you are explaining, and fresh questions whenever the class needs another round.
