Getting Started

Use this tool to build fluency from like terms to advanced multi-variable simplification.

Level Guide

  • Level 1. Like Terms: One variable and constants.
  • Level 2. Two Variables: Combine x and y terms with constants.
  • Level 3. Advanced Simplification: Products and powers such as xy, x²y, a², b², ab.

Common Mistakes

  • Combining unlike terms.
  • Sign errors when regrouping before combining.
  • Dropping variable parts when adding coefficients.
