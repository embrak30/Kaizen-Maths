Further Maths Vectors

Overview

This Further Maths tool extends vector work into 3D vector operations, scalar products, vector equations of lines, planes, cross products, and line-plane intersections.

Level Guide

  • Level 1: 3D position vectors, displacement vectors, magnitude, and unit vectors.
  • Level 2: Scalar product, angles, perpendicular vectors, and projections.
  • Level 3: Vector equations of lines, intersections, and shortest distance from a point to a line.
  • Level 4: Cross product, equations of planes, distance from a point to a plane, and line-plane intersections.

Teaching Focus

  • Use component notation consistently in 3D.
  • Use a · b = |a||b| cos θ for angles.
  • For a line, use r = a + λd, where a is a point and d is a direction vector.
  • For a plane, use n · r = n · a, where n is a normal vector and a is a point on the plane.
