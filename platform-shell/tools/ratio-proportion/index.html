Ratio and Proportion

Key Concepts

Ratios compare relative quantities. Proportion describes situations where quantities scale together or one changes as another changes.

Level Progression

  • Level 1: Simplifying ratios and sharing quantities in a given ratio.
  • Level 2: Scale maps, recipes, mixtures, and currency exchange.
  • Level 3: Direct and inverse proportional relationships.

Visual System

Known values are highlighted in blue.
Unknown values are highlighted in green.
Proportion setups are highlighted in yellow.

Core Principles

  • Equivalent ratios: multiply or divide each part by the same number.
  • Direct proportion: as one quantity increases, the other increases proportionally.
  • Inverse proportion: as one quantity increases, the other decreases proportionally.
  • Cross multiplication: a useful method for solving proportions.

Solution Framework

  1. Identify the relationship.
  2. Extract the known and unknown quantities.
  3. Set up the ratio or proportion.
  4. Calculate using unitary method, sharing, or cross multiplication.
  5. Check the answer in context.

Common Mistakes

  • Changing the order of quantities in a ratio.
  • Forgetting to simplify to lowest terms.
  • Confusing direct and inverse proportion.
  • Not checking whether answers are reasonable in context.

Using the Timer

  • Use 3 minutes for basic ratios.
  • Use 5 minutes for applied proportion scenarios.
  • Focus on method and interpretation rather than speed alone.
