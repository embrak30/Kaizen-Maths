Transformations Guidance

Overview

This tool supports transformations on coordinate grids: translations, reflections, rotations, enlargements, describing transformations, and combined transformations.

Teaching Focus

  • Use coordinates to make each transformation precise.
  • For rotations and enlargements, identify the centre before moving any point.
  • For reflections, every point and its image should be the same perpendicular distance from the mirror line.
  • For enlargement, use the vector from the centre to each vertex, then multiply it by the scale factor.
  • When describing a transformation, include every required detail: type, line or centre, angle, direction, scale factor, and vector where relevant.

Progression

  • Level 1: translate and reflect shapes using coordinate rules.
  • Level 2: rotate and enlarge shapes from a given centre.
  • Level 3: describe transformations fully from a diagram.
  • Level 4: use negative scale factors and combined transformations.
