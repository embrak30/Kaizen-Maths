Teacher Instructions

Overview

This tool extends basic integration into substitution and integration by parts. It is intended for A-Level Pure and Further Maths preparation where students must select a method and show the setup clearly.

Level Guide

  • Level 1: Indefinite integration by substitution.
  • Level 2: Definite substitution, including changing limits and logarithmic derivative forms.
  • Level 3: Integration by parts using the full choice of u, dv, du, and v.
  • Level 4: Mixed method selection between substitution and parts.

Teaching Notes

  • For substitution, show what u is, find du/dx, rewrite dx, substitute, integrate, and then return to x.
  • For definite substitution, either change the limits to u-values or return to x before substituting bounds.
  • For integration by parts, write the formula before substituting into it.
