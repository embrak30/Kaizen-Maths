Missing Angles Guidance

Overview

This tool practises finding missing angles from core angle facts and algebraic angle equations.

Level Guide

  • Level 1: Straight lines, triangles, and quadrilaterals.
  • Level 2: Mixed core angle facts.
  • Level 3: Straight-line, triangle, and quadrilateral angle facts with algebraic expressions.

Key Facts

  • Angles on a straight line add to 180 degrees.
  • Angles in a triangle add to 180 degrees.
  • Angles in a quadrilateral add to 360 degrees.

Common Misconceptions

  • Using 180 degrees for a quadrilateral instead of 360 degrees.
  • Forgetting to include every labelled angle in the sum.
  • Forgetting to solve for x before finding the requested angle.
