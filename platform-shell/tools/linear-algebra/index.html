Linear Algebra

Overview

This Further Maths tool builds linear algebra fluency through systems, row operations, span, basis, linear independence, linear transformations, kernel, image, rank-nullity, and change of basis.

Level Guide

  • Level 1: Systems of equations, augmented matrices, row operations, and triangular systems.
  • Level 2: Linear combinations, span, linear independence, bases, and dimension.
  • Level 3: Linear transformations, standard matrices, kernel, image, rank, and nullity.
  • Level 4: Change of basis, basis coordinates, and mixed linear map problems.

Teaching Focus

  • Keep vectors as column vectors so students distinguish vectors from coordinate points.
  • Stress that a basis must span the space and be linearly independent.
  • For kernel questions, solve Ax = 0 and write the solution as a span.
  • Use rank-nullity as a check: rank + nullity = number of columns.
