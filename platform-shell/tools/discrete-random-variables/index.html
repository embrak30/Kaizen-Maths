Teacher Instructions

Getting Started

This tool helps students master discrete random variables, expectation, variance, and common standard distributions.

Level Guide

  • Level 1: Probability distributions, missing probabilities, and validity checks.
  • Level 2: Expected value, variance, standard deviation, and linear expectation properties.
  • Level 3: Binomial, geometric, Poisson, and distribution-identification tasks.

Key Concepts

  • Expected Value: E(X) = sum of x times P(X=x).
  • Variance: Var(X) = E(X²) - [E(X)]².
  • Standard Deviation: SD(X) = sqrt(Var(X)).

Common Mistakes

  • Forgetting probabilities must sum to 1.
  • Not weighting values by their probabilities.
  • Using E(X²) - E(X) instead of E(X²) - [E(X)]².
  • Thinking Var(X + b) changes when only a constant is added.
