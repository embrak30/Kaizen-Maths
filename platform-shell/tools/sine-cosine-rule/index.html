Teacher Instructions

Overview

This tool supports teaching non-right triangles using the Sine Rule, Cosine Rule, and sine area formula.

Level Guide

  • Level 1: Find a side, find an angle, and find area using 1/2 ab sin C.
  • Level 2: Use the Cosine Rule for sides and angles, then combine rules in mixed questions.

Assessment Ideas

  • Identify the correct rule from the given data: AAS, ASA, SSA, SAS, or SSS.
  • Check appropriate rounding and units.
  • Discuss ambiguous cases for the Sine Rule after students master the standard cases.

Common Misconceptions

  • Confusing included and non-included angles.
  • Applying the Cosine Rule when the Sine Rule is more direct.
  • Rounding too early in multi-step problems.
  • Forgetting area units or degree mode on the calculator.
