Exponents and Indices

Getting Started

This tool builds index notation fluency from repeated multiplication through systematic use of index laws.

Level Guide

  • Level 1: Products in index form, evaluating powers, prime factorisation, and zero or negative indices.
  • Level 2: Same-base multiplication, same-base division, powers of powers, and mixed same-base practice.
  • Level 3: Complex multi-step simplification using several index laws.

Key Index Laws

  • Multiplication: am × an = am+n.
  • Division: am ÷ an = am-n.
  • Power of a power: (am)n = amn.
  • Zero index: a0 = 1 where a is not 0.
  • Negative index: a-n = 1/an.

Teaching Strategies

  • Emphasise that same-base laws require the same base.
  • Connect index notation to repeated multiplication before abstract laws.
  • Ask students to name the rule before calculating.
  • Use complex examples to practise one clean step at a time.

Common Mistakes

  • Adding indices for division instead of subtracting.
  • Multiplying indices when multiplying same-base terms.
  • Treating negative indices as negative values rather than reciprocals.
  • Forgetting that any non-zero base to the power 0 is 1.
