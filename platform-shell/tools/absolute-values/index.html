Absolute Values - Guide

Three-Level Structure

Level 1 builds meaning, Level 2 develops absolute value equations, and Level 3 introduces absolute value inequalities.

Level 1: Foundations

  • Basic numerical evaluation
  • Distance between two points on a number line
  • Evaluation of a single expression for negative, zero, and positive values

Level 2: Equations

  • E1: Pure term equations such as |x| = k and |3x| = k
  • E2: Pure term plus or minus an outside constant
  • E3: Linear expressions such as |ax + b| = k
  • E4: Linear expressions with an outside constant
  • Applications: Tolerance and acceptable range problems

Level 3: Inequalities

  • Absolute value inequalities using <, ≤, >, and ≥
  • Compound solutions for less-than cases
  • Split solution sets for greater-than cases
  • Number line interval notation included in worked steps

Key Teaching Points

  • Absolute value measures distance from zero, so outputs are never negative.
  • |A| = k creates two cases: A = k or A = -k.
  • |A| < k means the expression is between -k and k.
  • |A| > k means the expression is outside the interval from -k to k.

Common Mistakes

  • Forgetting to split equations into two cases
  • Using "and" instead of "or" for greater-than inequalities
  • Dropping the outside constant before isolating the absolute value
  • Treating |-x| as a negative number instead of a distance
