Getting Started

This tool builds fluency solving linear inequalities with clear algebraic steps and visual number lines. All math is rendered in KaTeX for clarity.

Level Guide

  • Level 1 — Basic Linear: One-step and two-step inequalities, including clean fractions and decimals.
  • Level 2 — Advanced Linear: Variables on both sides, grouping symbols, and flip-the-inequality cases.

What Students Will Practice

  • Solving one-step and two-step linear inequalities.
  • Flipping the inequality when multiplying or dividing by a negative.
  • Graphing solutions on a number line.
  • Writing solutions as inequalities and intervals.

Common Student Mistakes

  • Forgetting to flip the sign when multiplying/dividing by a negative.
  • Using closed dots for strict inequalities or open dots for inclusive inequalities.
  • Dropping the variable when rearranging both sides.
