Bar Models

Purpose

Use this display tool to make hidden relationships visible before moving to formal written methods.

How to Teach With It

  • Display one example and ask pupils what each bar or part represents.
  • Reveal one step at a time so the class predicts the next move.
  • Use the final step to connect the model to the calculation.

Useful Prompts

  • What does one part represent?
  • Which quantity is the whole?
  • What has been added, removed, shared, or compared?
  • How does the bar model match the calculation?

Common Misconceptions

  • Using the final amount as the original whole in reverse percentage problems.
  • Sharing a ratio by the number of groups instead of the total number of parts.
  • Treating each bar as equal when a comparison model shows unequal quantities.
