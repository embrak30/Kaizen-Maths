Loci and Constructions Guidance

Overview

This tool supports GCSE-style construction and loci work using clear compass-and-straightedge diagrams, boundary lines, and worked construction steps.

Teaching Focus

  • Separate the given information from the construction marks.
  • Keep the compass width fixed when constructing perpendicular bisectors and angle bisectors.
  • Emphasise that a locus is a complete set of points satisfying a condition, not just one point.
  • For regions, construct each boundary first, then decide which side or part satisfies the condition.

Progression

  • Level 1: standard constructions using arcs and straight lines.
  • Level 2: core loci from points, lines, pairs of points, and pairs of lines.
  • Level 3: two-condition regions and intersection problems.
