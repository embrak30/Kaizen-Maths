Teacher Instructions

Getting Started

This tool helps students practise finding intersection points of conic sections with lines. Each set contains two longer challenge problems with worked solutions.

Level Guide

  • Level 1 - Circles: Circle-line intersections including secants, tangents, and no-intersection cases.
  • Level 2 - Parabolas: Parabola-line intersections by substitution.
  • Level 3 - Hyperbolas: Hyperbola-line intersections and asymptote-adjacent cases.
  • Level 4 - Mixed: Mixed conics and special cases.

Core Method

  • Substitute the line equation into the conic equation.
  • Simplify to a quadratic in one variable.
  • Use the discriminant to decide how many intersections exist.
  • Substitute each x-value back into the line to find y.
