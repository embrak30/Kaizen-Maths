Teacher Instructions

Purpose

This tool uses a Kaizen-owned training data set so teachers can practise large-data-set skills without copying an exam-board data file.

Dataset Context

Teachers can choose from four Kaizen-owned training data sets: weather and climate, school attendance, transport and travel, and health and fitness. Each context has variables, rows, summaries, relationships, and interpretation prompts.

Level Guide

  • Level 1: Understand variables, units, rows, filters, missing values, and outliers.
  • Level 2: Use summary statistics to compare groups and interpret averages and spread.
  • Level 3: Interpret graph choices, correlation, regression, and predictions.
  • Level 4: Mixed exam-style interpretation with limitations and conclusions in context.

Teaching Notes

  • Students should refer to the context in every written conclusion.
  • Emphasise units and whether a variable is continuous, discrete, categorical, or derived.
  • For comparison questions, students should quote evidence and comment on both centre and spread when relevant.
