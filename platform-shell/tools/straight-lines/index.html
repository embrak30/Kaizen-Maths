Teacher Instructions

Getting Started

This tool helps students practise straight-line concepts in a structured progression from gradient to line relationships.

Level Guide

  • Level 1 - Gradient: Find gradient from two points and from equations.
  • Level 2 - Intercepts: Find x-intercepts, y-intercepts, or both.
  • Level 3 - Equations: Find equations from points, gradients, and intercepts.
  • Level 4 - Intersections: Find intersections and parallel or perpendicular lines.

Key Formulas

  • Gradient: m = y2 - y1x2 - x1
  • Equation forms: y = mx + c and ax + by + c = 0.
  • Parallel lines: Same gradient.
  • Perpendicular lines: Gradients multiply to -1.
