Teacher Instructions

What this tool is for

This tool provides practice solving trigonometric equations in radians on the interval 0 <= x < 2pi, including equations that simplify to quadratics in sine or cosine.

Level Overview

  • Level 1: Linear equations in sin(kx + phi), cos(kx + phi), and tan(kx + phi).
  • Level 2: Quadratics in sin(kx + phi) or cos(kx + phi), factorised and fully solved.

Teaching Strategies

  • Isolate the trigonometric expression first, then substitute theta = kx + phi.
  • Use the unit circle, symmetry, and special angles to generate all solutions.
  • For quadratics, solve in a substitute variable before returning to the trig equation.
