TEACHER INSTRUCTIONS
Teacher Instructions

Purpose

This tool generates linear programming practice with feasible-region diagrams, vertices, objective functions, and contextual optimisation problems.

Level Guide

  • Level 1: Read constraints, identify feasible regions, list vertices, and test points.
  • Level 2: Maximise or minimise an objective function by testing vertices.
  • Level 3: Interpret linear programming in context, including profit, cost, and integer solutions.

Teaching Notes

  • The optimum of a linear objective over a polygonal feasible region occurs at a vertex.
  • Students should list all vertices before testing the objective function.
  • For integer contexts, check lattice points when the best continuous vertex is not a whole-number solution.
  • Worked steps show constraints, vertices, objective values, and the final interpretation.

Common Mistakes

  • Shading the wrong side of an inequality boundary.
  • Testing only one vertex instead of every feasible vertex.
  • Forgetting non-negativity constraints such as x ≥ 0 and y ≥ 0.
  • Giving a coordinate answer without interpreting what x and y mean in context.
