Free Vectors Guidance

Overview

This tool builds fluency with free vectors using column vector notation only: adding and subtracting vectors, multiplying by a scalar, and finding vector length.

Level Guide

  • Level 1: Add and subtract column vectors.
  • Level 2: Multiply vectors by scalars and mix the core operations.
  • Level 3: Find the length of a vector and the length of resultants.

Key Ideas

  • A column vector records horizontal movement first, then vertical movement.
  • Vector addition and subtraction are component-wise.
  • Multiplying by a scalar multiplies every component.
  • The length of vector (a, b) is sqrt(a^2 + b^2).

Common Misconceptions

  • Writing vertical movement before horizontal movement.
  • Losing the sign when movement is left or down.
  • Only multiplying one component by a scalar.
  • Using a + b instead of Pythagoras for vector length.
